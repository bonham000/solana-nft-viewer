## Local Development

Clone the repo and run `yarn start`. These additional commands are also available:

```sh
# Run an app build
$ yarn build

# Run TypeScript compiler
$ yarn tsc

# Run Prettier checks
$ yarn prettier

# Run ESLint checks
$ yarn eslint

# Run unit tests
$ yarn test

# Run all checks
$ yarn test:all
```
